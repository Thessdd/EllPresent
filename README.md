# EllPresent ✦

Single-page “maximalist tender chaos” per il 26° compleanno di Ell — sorpresa da Camilla.

## Setup (locale)

Serve Node.js (consigliato 20+).

```bash
npm install
npm run dev
```

Build:

```bash
npm run build
npm run preview
```

## Deploy (Vercel)

Progetto Vite statico. Il `vercel.json` include la rewrite richiesta.
